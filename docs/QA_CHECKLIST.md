# QA CHECKLIST (Manual Smoke Test)

## 0. 前提
- iPad Swift Playgrounds で `Kyounani.swiftpm` を起動。
- テスト対象データが無い場合は親モードで数件作成してから実施。

## 1. Today
- [ ] 起動直後にTodayが表示される
- [ ] 「きょう / つぎ / あした」セクションが表示される
- [ ] 子どもフィルタ（息子/娘/両方）が切り替わる
- [ ] 「つぎ」タップで日本語読み上げ + TimerRing が動く
- [ ] 予定0件時の空状態UIが崩れない

## 2. Calendar / DayDetail
- [ ] Today/Calendarタブを往復できる
- [ ] 月/週切替ができる
- [ ] 日セル最大2件 + `+N` 表示が出る
- [ ] 日別詳細へ遷移できる
- [ ] 日別詳細で予定タップ時に読み上げ/リング動作

## 3. 親ゲート
- [ ] 右上領域3本指2秒長押しでゲート起動
- [ ] 正しい4点シーケンスで解除できる
- [ ] 連続失敗でクールダウンが発生する
- [ ] 親モードで「ロック」後に子どもモードへ戻る

## 4. CRUD
- [ ] 親モードで新規予定を作成できる
- [ ] 一覧から編集できる
- [ ] 削除できる
- [ ] 空タイトル時に既定文言で保存される

## 5. 繰り返し例外
- [ ] 週次繰り返し予定を作成できる
- [ ] 「この日だけ / 以降 / 全体」を選択して編集できる
- [ ] 「この日だけ削除」で確認ダイアログが出る
- [ ] 影響範囲プレビュー（次3回＋概算件数）が表示される

## 6. スタンプ追加
- [ ] Filesから追加できる
- [ ] Photosから追加できる
- [ ] 追加後にEventEditorで選択できる

## 7. バックアップ
- [ ] `.kybk` を書き出せる
- [ ] 正しいパスフレーズで復元できる
- [ ] 誤パスフレーズ時に既存データが壊れない

## 8. 回帰観点
- [ ] 子どもモードで編集導線が露出しない
- [ ] テーマ切替（Kid/High Contrast）がToday/Calendar/DayDetailに反映される
